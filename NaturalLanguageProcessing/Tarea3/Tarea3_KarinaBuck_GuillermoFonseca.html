def tokenizacion(message):
  '''Devuelve un diccionario con el mensaje tokenizado 
     convirtiendo el mensaje a minusculas, utilizando TweetTokenizer 
     (ya que la base de datos posee tweets), eliminando stopwords y puntuacion
     para medir mejor la similitud
     El mensaje a recibir debe ser un string'''
  # Usamos tweet tokenizer
  # Ademas convertimos todo a minusculas
  tokens = tweet_tokenizer.tokenize(message.lower())

  # Eliminamos stopwords
  tokens = [x for x in tokens if x not in stopwords]
  # Eliminamos puntuacion
  tokens = [x for x in tokens if x not in puntuacion]
  return tokens

Medicion de similitud¶

Ahora se medira la similitud entre los documentos del corpus:

  1. Creamos la variable similitud haciendo el producto punto en una lista por comprensión que se recorren dentro de nuestro corpus.
  2. Creamos la matriz similitud, donde es simétrica y su diagonal está compuesta por ceros.
  3. Para poder medir similitud entre textos, se escoge hacerlo a través de "aritmetica modular", ya que numpy para buscar el valor máximo de una matriz, la convierte en array unidimensional y al buscar el valor máximo, entrega la posición de ese array unidimensional.

def entrega_max_similitud(similitud_matrix, num_max=2):
  ''' Funcion que, dada una matriz de similitud tridiagonal superior con diagonales iguales a 0,
      imprime un numero de maximos igual a la cantidad de num_max
      y entrega el mayor de ellos '''
  array_1d = similitud_matrix.flatten()
  max_indexes = array_1d.argsort()
  n,_ = similitud_matrix.shape
  filas = max_indexes[-1:-(num_max + 1):-1]%n
  columnas = max_indexes[-1:-(num_max + 1):-1]//n
  max_values = np.zeros(num_max)
  print("Las similitud maximas son de:")
  for i in range(filas.shape[0]):
    max_values[i] = similitud_matrix[columnas[i]][filas[i]]
    print(max_values[i], f"de los documentos {columnas[i]} y {filas[i]}")
    print("Correspondientes a:")
    print(df_documents.values[columnas[i]])
    print(df_documents.values[filas[i]])
  return max_values[0], columnas[0], filas[0]

Vectorización TF-IDF¶

Comenzamos la vectorización viendo el léxico para este corpus que contiene 50 elementos (noticias):

  1. Primero enlistamos los documentos y sus tokens
  2. definimos la variable de todos los tokens en los documentos
  3. Definimos el léxico, utulizamos la función "sorted" para que nos devuelva el objeto en una lista ordenada
  4. Importamos la funcion "OrderedDict" de la librería "collections" para creaar un diccionario con los tokens y ceros.
  5. Creamos el vector td-idf como lista vacía y comenzamos a recorrer cada noticia dentro del corpus, comenzando con la creación de una matriz de ceros.
  6. Ingremos nuevamente a recorrer, esta vez recorremos cada elemento del diccionario.
  7. Finalmente se aplican las formulas correspondientes a TF e IDF

Tf(t,d)=f(t,d)/|L_d|

IDF(t,C)=|C|/|{ #d : t \in d}|

Calculo de similitud¶

Para el cálculo de similitud, comenzamos importando la librería math, que trae consigo la función sim_coseno la cúal toma dos vectores y calcula su similitud. Creamos la lista por compresiónpara luego calcular elproducto punto. Finalmente retoranmos la el % de similitud de los textos más similiares.

import math

def sim_coseno(vec1, vec2):
    vec1 = [val for val in vec1.values()]
    vec2 = [val for val in vec2.values()]
    
    dot_prod = 0
    for i, v in enumerate(vec1):
        dot_prod += v*vec2[i]
        
    norm_1 = math.sqrt(sum([x**2 for x in vec1]))
    norm_2 = math.sqrt(sum([x**2 for x in vec2]))
    
    return dot_prod / (norm_1 * norm_2)
